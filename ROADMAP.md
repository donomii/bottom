# Roadmap

## Current implementation

- Trustworthy process lifetimes and stable identities.
- Direct Linux lifecycle events with explicit coverage gaps.
- Native Linux connector, Windows ETW, and entitlement-gated macOS Endpoint Security lifecycle events with explicit polling fallback.
- Native Linux, macOS, and Windows snapshots.
- Readable process start, exec, and stop logs.
- Immediate-key, resize-aware TUI navigation with selectable columns and ordering.
- Cross-platform CI with native lifecycle smoke checks and automation for checksummed, SBOM-described, provenance-attested release archives.
- Public Homebrew tap and Scoop bucket recipes.
